Power Toybox Deck is an unofficial Stream Deck plugin compatible with Microsoft PowerToys. It mirrors your local PowerToys setup so a single configurable action can launch utilities, global shortcuts, PowerToys Run aliases, Command Palette commands, and Keyboard Manager mappings.

Each Stream Deck key stores a stable command ID instead of a copied shortcut. When you change a PowerToys hotkey, enable or disable a module, update a Run alias, or adjust Command Palette entries, configured keys refresh their title, icon, and execution behavior automatically.

Requirements: Windows 10 or Windows 11, Stream Deck 6.9 or newer, Node 20 runtime support from Stream Deck, and Microsoft PowerToys installed for the current Windows user.
